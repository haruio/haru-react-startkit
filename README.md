README

## Quick start

1. git clone https://github.com/haruio/haru-react-startkit  
1. npm install or yarn install
1. npm start or yarn start

## Script

1. npm run dev
1. npm run test
1. npm run build

1. react
1. react-router
1. redux

1. immutable
1. classnames
